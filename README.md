
# To Install and Run the Project

You must have dfx installed in your local environment!!!



1. start local dfx

```
dfx start --clean
```


2. Deploy canisters

```
dfx deploy

```

3. Run NPM server

```
npm start
```

4. Head to localhost link 

click on the local link after npm starts that looks like this -  "http://localhost:8080/".
